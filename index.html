Recent Projects

Pediatric Hospital Interface

Java, JavaFX
Over the course of one semester at ASU me and four of my peers created a database meant for a pediatric hospital. This database was full equipped with password protected account creation, messaging systems between the caretakers and patients, and admin capability for data hiding. The backend was done using Java and the front end was done in JavaFX.

Graphical Interface of COVID-19 Data

Java, JavaFX, SQL
This was a group project meant to display the the data me and three of my peers collected on COVID-19 trends in South American countries. First we used one of our virtual machines to host an SQL database. We then uploaded the raw numbers into the database and created queries written in PostgresSQL to pull data from the database at will. We used Java to connect to the database and JavaFX to display the queries in graph form.
